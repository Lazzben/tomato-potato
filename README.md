# Tomato-Potato
## 项目简介
该项目将番茄工作法与代办事项列表结合了起来，用相关图形直观展示最近的工作状态与内容，是一款简单易行的时间管理软件。

预览地址：

技术栈：React | Redux | TypeScript | Sass | date-fns | lodash | axios
